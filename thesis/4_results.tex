\section{Hauptteil}

Der folgende Programmcode ist nicht repräsentativ für das Ergebnis einer
erfolgreichen Abschlussarbeit.

\begin{lstlisting}
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
\end{lstlisting}
